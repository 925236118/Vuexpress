# Vuepress

Vue + Express 前端模板
## 使用技术
- Vue
- Vue-router

## 目录
```
.
├─node_module
├─public
└─src
    ├─assets
    ├─components
    ├─config
    ├─router
    ├─service
    │  ├─api
    │  └─request
    ├─store
    │  └─modules
    └─views
        ├─Chat
        ├─Error
        └─Main
```

## 启动
```
npm install
npm run dev
```
